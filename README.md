# Terra

A server bootstrapping package for personal use.

Makes use of [tj/stack](https://github.com/tj/stack) for provisioning.

## Getting Started

```bash
wget -qO- https://raw.githubusercontent.com/rosszurowski/server/master/bootstrap.sh | sh
```
